70.2.Graphic line

It allows to draw a frame around the selected items.

This tool is available in the "Map / Graphics / Graphic line" menu and the relevant button bar.

A dialog box where you can define the properties of the line graph will be displayed:

  • Place options. It allows to select the place of the graphic line.

    • Around the graphics selected

    • Around all the graphics. Not required to have any selected graph

    • With reference to margins. Not required to have any selected graph

  • Group graphic line with graphics. With the box checked allows you to group the graphic line and the element, so as to form a single entity and not two independent.

  • Position of the line. It allows to configure the position of the graphic line

  • Configure. It allows to access to the simbology selector

70.3.To Align

It allows to change the alignment, distribution and size of the selected items.

This tool is available from the "Map / Graphics / Align" menu.

A dialog box where you can set options to align items will be displayed

  • Alignment. It allows to align the selected elements, placing them on the same axis: left, center, right, horizontal, top, center and bottom vertically.

  • Distributing. It allows to place the elements at equal intervals in the area defined by the selected items.

  • Match size. It allows to modify the size of a selected item with reference to another selected item. The adjustments are made based on the largest element in the selection there. The size of the elements can be matched in width, height or both directions respectively.

  • Space. It allows to do a spatial distribution of the selected items.

  • In the layout. The tool is applied to the entire area of the map.
